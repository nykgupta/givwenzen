[[!-GivWenZen-!][.FrontPage]]

 * To use [[GWZ][.FrontPage]] you need to add the givwenzen.jar and the dependencies below.

 * To use [[GWZ][.FrontPage]] with the slim testing system use the !-org.givwenzen.GivWenZenForSlim-! by adding the following table to a tests.
-----
|import|
|org.givwenzen|

|script|
|start|domain steps for slim|
-----

 * Add a step to the test created in step 2.
-----
|given|this is a simple step|
-----

 * Create a [[step class][.UserGuide.StepClases]] is the default step package bdd.steps.
----
{{{
@DomainStep("this is a simple step")
public boolean simpleStep() {
  return true;
}
}}}
----

 * Run the test
